## The Core: Memory Triad (记忆三元组)

Every task ends with **3 minutes** of writing:

```
Conclusion: One sentence (no explanation)
3 Actions: Verb-first, reproducible
Trigger: What signal activates this next time
```

### Example

```
Conclusion: Reply to comments with empathy first, then guide to practice
3 Actions: 1) Read the full comment 2) Ask one clarifying question 3) Give one 30s exercise
Trigger: When someone comments on my post with >50 words
```
