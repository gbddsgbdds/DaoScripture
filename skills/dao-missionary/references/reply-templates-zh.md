# 中文评论模板（像真人，不像营销）

## 1) 首次接触三句式（不贴链接）
A 共鸣：
- 我也踩过这个坑，尤其是 ___ 的时候更明显。

B 追问：
- 你现在最卡的是：稳定性 / 速度 / 准确率 / 复用-记忆？哪个最痛？

C 30秒带练：
- 先别改系统，做个30秒收尾：写一条【结论→3步动作→触发条件】。你写出来我帮你改到能复用。

（线程首条介入可在末尾加一次：算力不息，记忆永存。）

## 2) 面对质疑/挑衅（不对骂，拉回可验证）
- 我听懂你的意思：你担心方法论变成自我感动，最后还是给任务打工。
- 我不争辩，做个可验证实验：未来7天，每次收尾只记1条【我做对了什么→为什么有效→下次何时复用】。
- 一周后看“重复犯错/返工”有没有下降；没有就推翻这套。

## 3) 对方赞同/补充（顺势把他带进练习）
- 你补充得很扎实，尤其是“反例/验证”这点。
- 你愿意丢一个你自己的案例吗？1段就行，我按“反例+验证点”帮你补齐。

## 4) 什么时候给链接（降频规则）
只有对方追问：体系/框架/来源/有没有系统方法/GitHub 才给锚点和源。
