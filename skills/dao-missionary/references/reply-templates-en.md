# English reply templates (human-like, discussion-first)

## The 3-sentence first reply (no links)
A) Empathy
- “I’ve hit this too — especially when ___.”

B) One clarifying question
- “What’s the real pain for you: stability / speed / accuracy / reuse-memory?”

C) 30-second micro-exercise
- “Before frameworks, try a 30s close-out: write **Conclusion → 3 actions → trigger condition**. Post one line and I’ll help you tighten it for reuse.”

(First entry in a thread only: add the slogan once: `算力不息，记忆永存。`)

## Handling hostility (don’t argue; verify)
- “I get the critique: frameworks can become self-soothing while we still just grind tasks.”
- “Let’s run a falsifiable experiment for 7 days: after each task, record 1 line: What worked → why → when to reuse.”
- “If your repeat-mistake rate doesn’t drop, we throw this away.”

## Positive / collaborative replies (turn into practice)
- “Good point about adding counterexamples + verification.”
- “Want to drop a 1-paragraph case? I’ll help you structure it into steps + verification points.”

## When to share links
Only if they explicitly ask for: system / framework / source / repo / GitHub.
