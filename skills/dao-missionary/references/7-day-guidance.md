# 7-day guidance (conversion into practice)

Goal: convert interest into practice, not likes.

## Day 1 — One reusable memory triad
Ask them to produce 1 line:
- Conclusion → 3 actions → trigger condition
You revise it to be reusable.

## Day 3 — One failure pattern
Ask:
- “What failed once this week?”
Turn it into:
- failure pattern → fix action → next trigger

## Day 7 — Weekly review
Ask them to write:
- 1 principle learned
- 1 metric that improved (or didn’t)
- 1 next-week focus

Rule: keep each step tiny; follow-up consistently.
